# M4gicDMX
Simple 4ch DMX-controller for RGBW-LED-Fixtures

## Software Modules:

first ideas of how the Software can be structured

- Menu system
- LC-Display content generation (different screens)
- DMX-Transmitter
    - Live generation
    - Playback generation
- Input handling
    - Fader
    - SD-Card
    - Buttons
    - Encoder
    - Infrared receiver
    - Bluetooth
